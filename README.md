# GitHub Explorer  

📱 A React Native mobile application to explore GitHub repositories, view detailed information, and manage favorites.  

## ✨ Features  

- 🔍 **Search GitHub Repositories** by name  
- 📊 View **repository details**, including:  
  - 🌟 Stars, 🍴 Forks, 👀 Watchers  
  - 🏷️ Primary programming language  
  - 📅 Created & Last Updated dates  
  - 🔖 License Information  
  - 🌲 Default Branch  
- 👥 See **Top Contributors** for a repository  
- ❤️ **Save Favorite Repositories** for quick access  
- 🔗 Open repositories directly in your browser  
- 🎨 Clean & Intuitive UI using `StyleSheet`  

## 🛠️ Installation  

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/santhosh46v/GitHubExplorer.git
2. **Navigate to the project directory:**
   cd GitHubExplorer
3. **Install dependencies:**
   npm install
    or
   yarn install
4. **Run the app using Expo:**
   npx expo start

## 📦 Dependencies

• React Native (Expo)
• React Navigation
• Axios (for GitHub API calls)
• AsyncStorage (for local favorites storage)
• React Native Vector Icons
